##### what is this?
its a `.css` file that tweaks the visuals of the youtube music desktop application. works by injecting custom styling rules that override the default colors and appearance of buttons, backgrounds, text, and other elements in the app interface.

##### credit
the base template for this theme was originally created by [oceanicsquirrel](https://github.com/OceanicSquirrel/themes-for-ytmdesktop-player). i have modified stuff here and there but mostly the credit goes to them.

<br>

<h4>installation instructions</h4>

use this theme with the [peardesktop yt music app](https://github.com/pear-devs/pear-desktop) (by pear-devs).

##### pear desktop
- **clone the repository:**
```sh
git clone https://github.com/seivarya/yt.themes 
```
- **apply the theme:**
- open the application settings (gear icon or options menu).
- go to **visual tweaks**.
- select **themes**.
- choose the option to load or select a custom file, and navigate to the cloned `.css` file.

##### add your own theme!

each theme gets its own directory unless the theme is part of a palette. for example, `tokyo_night_moon.css` belongs to the tokyo_night/ palette.
